Alcohol and Tobacco Tax and Trade Bureau
Label Verification System
Prototype

Verify a label

Add the label image. A single front label is fine — but if the warning, net contents, or bottler appear on a separate back label, add that too and they'll be checked together.

Compare against application data (optional)

In production these values come from the COLA application automatically. Enter them to also compare the label field-by-field, or leave blank to check the label for compliance only.

Verify a batch of labels

For high-volume submissions — select multiple label images at once and verify them together.

    Application data manifest (optional)

    To compare each label against its COLA values, provide a JSON object mapping each file name to its application fields. Leave blank to check every label for compliance only.

    Prototype for evaluation. Standalone — not connected to COLA. No label data is retained.
